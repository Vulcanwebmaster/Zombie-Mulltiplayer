Jouer a Zombiz! Team Survival

Découvrez vos ennemis :

Le zombiz

Description : Pas très malin, le Zombiz, c'est la base, la lada des zombies. Si jamais ils arrivent à vous attraper, vous êtes probablement obèse, ou endormi. Bref, c'est gratuit, ça prend bien les balles, que du bonheur !
Attaque : Très faible
Vitesse : Faible
Endurance : Assez Faible
Abilités : Aucune

Le sprinter

Description : Bien plus rapide que les zombiz, les sprinters sont des adversaires difficiles à battre... Les 2 premières minutes. Ils vous donneront tout de même un léger surplus de difficultés, d'autant plus qu'ils sont plus endurants que les Zombiz. A vos fusils, prêt, tirez !!
Attaque : Plutôt faible
Vitesse : Moyenne
Endurance : Faible
Abilités : Aucune

L'araignée

Description : Si vous avez déjà joué à Zombiz!, alors vous connaissez les araignées. Je dirais même que vous les détestez probablement. Rapide, collante, nombreuses... Vous les connaissiez à peine qu'elles étaient déjà vos pires ennemies. Rien ne sert de courir, Elles finiront toujours par vous rattraper... Brrr. Je déteste les araignées !
Attaque : Très faible
Vitesse : Très rapide
Endurance : Très Faible
Abilités : Aucune

Bouboule

Description : Vous l'aurez deviné, Bouboule ne fait pas dans la finesse. Une chance pour vous, il est du genre très lent. Mais diantre, qu'est ce qu'il est resistant !
Attaque : Moyenne
Vitesse : Très Faible
Endurance : Moyenne
Abilités : Aucune

M. boucher

Description : Le boucher aime son travail. Tellement, qu'il ne s'arrête jamais de travailler. Il est dans son monde, tant que vous ne l'approchez pas trop, il vous laissera tranquille. Méfiez-vous de son couteau, plus d'un aventurier a été servi dans son fameux plat de cervelle farcie !
Attaque : Très forte
Vitesse : Rapide
Endurance : Moyenne
Abilités : Aucune

Le soldat

Description : Vous vous demandiez ce qui pouvait être pire qu'un zombie avec un hachoir de cuisine ? Et bien, imaginez un fanatique ex-soldat. Le tout, en gilet par balle, avec le casque et tout le tintouin... Et oui. Le soldat, c'est ça. Autant vous dire que vous allez en gaspiller des balles pour en venir à bout...
Attaque : Moyenne
Vitesse : Moyenne
Endurance : Très résistant
Abilités : Aucune

Maîtresse des araignées

Description : Mmmh. J'ai bien peur que vous ayez ici un zombie de qualité. La puissance du boucher (en mieux !), la résistance du soldat (en mieux !) et la vitesse des araignées... En mieux ! Autant vous dire qu'avant de la reveiller, il faudra bien vous organiser. (astuce : faites tanker celui qui court le plus vite - zombiz slayer)
Attaque : Très forte
Vitesse : Très rapide
Endurance : Très résistante
Abilités : A venir...
